Named Colors: CSS supports a set of predefined color names, such as red, blue, green, black, white, etc.

Hexadecimal Notation: Colors can also be specified using hexadecimal notation, which consists of a # followed by six hexadecimal digits representing the red, green, and blue components of the color.

RGB (Red, Green, Blue): Colors can be defined using the RGB model, specifying the intensity of red, green, and blue components.

RGBA (Red, Green, Blue, Alpha): RGBA is similar to RGB but with an additional alpha parameter representing the color's transparency.

HSL (Hue, Saturation, Lightness): HSL is a cylindrical color model that specifies colors based on their hue, saturation, and lightness.

HSLA (Hue, Saturation, Lightness, Alpha): Similar to HSL but with an additional alpha parameter for transparency.

color: red;

Green (Named)

background-color: green;

Red (Hexadecimal)

Green (Hexadecimal)

Red (RGB)

color: rgb(255, 0, 0);

Green (RGB)

Semi-transparent Red (RGBA)

color: rgba(255, 0, 0, 0.5);

Semi-transparent Green (RGBA)

background-color: rgba(0, 255, 0, 0.7);

Red (HSL)

color: hsl(0, 100%, 50%);

Green (HSL)

background-color: hsl(120, 100%, 50%);

Semi-transparent Red (HSLA)

background-color: green;

Semi-transparent Green (HSLA)

background-color: hsla(120, 100%, 50%, 0.7);
